# Programming 3 — Exam Tasks & Solutions

> Faculty of Information Technologies, University "Džemal Bijedić", Mostar  
> Course: Programming III | C# · .NET · WinForms · Entity Framework

This repository contains exam tasks and solutions from the Programming 3 course. Each folder includes the original task PDF and a working Visual Studio solution.

---

## ⚠️ Before Running Any Project

1. Open the `.sln` file with **Visual Studio 2022** (right-click → Open with → Visual Studio 2022)
2. Do a **Rebuild Solution** (`Build → Rebuild Solution`) to restore all NuGet packages
3. Make sure the database connection string matches your local setup

> Solutions use **Entity Framework** with a local SQL Server database. The database is included in the project — rebuild is required before first run.
